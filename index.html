Various Commands for git

git status

git add (file name) / -A

git log

git commit -m "Added a file"

git show (commit id)

git checkout (commit id) -- (file name)

git checkout master -- (file name)

git restore (file name) -> undo written code before git add

git restore --staged (file name) -> undo written code after git add

git reset --hard/soft HEAD~1 -> hard for completely deleting after commit soft for editing the commited code

git log -p -2 -> (for last two commits)

git log --stat -> (summary of changes)

git log --pretty -> (show in a single line)

git log --pretty=format:%h -%an, %ar:%s
